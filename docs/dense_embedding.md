# 尝试引入稠密语义特征以提升泛化能力

## 动机
考虑到 TF-IDF 在处理词汇表外词汇和同义词泛化时的局限性，我们假设引入预训练的句子向量（Sentence Embeddings）可以捕捉更深层的语义信息，从而识别出那些难以通过字面特征识别的谣言样本。

## 实施细节
- 引入了轻量级语义模型 all-MiniLM-L6-v2。
- 使用 FeatureUnion 将原本的 TF-IDF 稀疏特征与 384 维的稠密语义特征进行了拼接。
- 为了防止量纲差异，后续在 Pipeline 中加入了 MaxAbsScaler 进行预处理。

## 实验结果
- 准确率从原本的 0.8803 降至 0.8579，宏平均 F1 分数也略有下降。

- 经过对比发现，Dense Embedding 的引入并没有带来预期中的增益，反而引入了额外的噪声，导致模型对谣言的捕捉能力（召回率）没有显著改善。


## 结论与反思

- 本项目中的谣言主要依赖于特定的词汇特征，比如突发事件的关键词和特定的推文结构，而非广泛的语义语境。

- TF-IDF 已经能够非常精确地捕捉到这些强信号。引入深度学习模型不仅增加了计算开销，运行速度变慢，还使得线性分类器难以在混合后的高维空间中进行最优切分。

- 该方案暂不合并到 main 分支，作为实验记录保留。
