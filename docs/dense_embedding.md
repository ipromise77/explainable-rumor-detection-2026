# 稠密语义特征探索记录

## 动机

TF-IDF 对短文本中的显式关键词和字符片段很有效，但对同义表达、语义相近表达的泛化能力有限。因此我们曾尝试引入预训练 Sentence Embedding，观察稠密语义特征是否能补充 TF-IDF 的不足。

## 实施细节

- 引入了轻量级语义模型 all-MiniLM-L6-v2。
- 使用 FeatureUnion 将原本的 TF-IDF 稀疏特征与 384 维的稠密语义特征进行了拼接。
- 为了减少量纲差异，后续在 Pipeline 中加入了 MaxAbsScaler 进行预处理。

## 实验结果

- 本地 TF-IDF 集成基线 Accuracy 为 0.8803。
- 稠密语义特征融合后 Accuracy 约为 0.8579，低于基线。
- 该方案还引入额外依赖、模型下载和更长运行时间，不利于课程项目复现。

## 结论

- 本数据集中的许多强信号来自事件关键词、hashtag、短语和局部字符模式，TF-IDF 已经能较好捕捉。
- 稠密向量并未在当前数据规模和模型设置下带来收益，反而可能引入噪声。
- 因此该方案只作为探索性负面结果记录，不作为最终主模型的一部分。
