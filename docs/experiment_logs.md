# 实验与调优记录

## 1. 动态阈值扫描

最初使用默认的 0.5 作为逻辑回归的判定阈值。分析混淆矩阵时发现基座模型偏向保守，漏报了较多谣言样本。我们编写了阈值扫描脚本，在 [0.20, 0.80] 区间内搜索最优阈值。结果显示 0.54 对漏报的校准效果最好，但准确率提升幅度极小，不显著。因此最终保留 0.5 默认阈值。

## 2. Event 特征融合实验（未成功）

考虑到谣言具有事件聚集性（相同突发事件会有相似的表述模式），我们尝试利用数据集中的 `event` 字段。对 `event` 做 One-Hot 编码后拼接到 TF-IDF 稀疏矩阵中。验证集准确率从 0.8803 小幅下降到 0.8753。

分析原因：词级与字符级 TF-IDF 提取了数万维特征，而事件特征仅有个位数维度。在训练时，事件维度的权重被文本特征完全淹没，不仅没有发挥作用，反而导致了对部分训练样本的轻微过拟合。

## 3. L1 正则化实验（未成功）

特征工程阶段，由于提取了近 8 万维特征，其中可能包含大量噪声。我们尝试将逻辑回归的默认 L2 正则化替换为 L1 正则化（Lasso），希望将无用特征的权重压缩为 0。

实际实验中，L1 正则化需要使用特定求解器（liblinear 或 saga），在近 8 万维的高维稀疏特征下训练时间明显增长，且出现收敛困难。考虑到课程项目对复现稳定性的要求，最终保留了默认的 L2 正则化方案。
